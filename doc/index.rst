**pyGEDAI**
===========

GEDAI in python


.. image:: _static/img/logo.png

Generalized Eigenvalue De-Artifact Instrument (GEDAI) is a method to remove artifacts from EEG/MEG data based on generalized eigenvalue decomposition.
