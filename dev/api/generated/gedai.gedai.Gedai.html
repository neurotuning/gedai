Parameters:
wavelet_typeint

Wavelet to use for the decomposition. The default is â€˜haarâ€™. See pywt.wavedec() for complete list of available wavelet values.

wavelet_levelint

Decomposition level (must be >= 0). The default is 0 (no decomposition). If 0 (default), no wavelet decomposition is performed. See pywt.wavedec() more details.

If float, zero out all bands whose upper frequency bound is below this cutoff frequency (in Hz). If None, no frequency band is zeroed out. The default is None.
